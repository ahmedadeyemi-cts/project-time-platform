# Module 042.3A — Popup-Free Invoice PDF Download

## Purpose

Remove the browser pop-up dependency from invoice PDF output.

## Behavior

- PDF output is downloaded from the authenticated immutable-invoice document endpoint.
- No `window.open()` or automatic browser print window is used.
- Users open the downloaded PDF with their normal browser or PDF viewer to print it.
- Engineer, Project Manager, and Project Coordinator privacy controls remain unchanged.
- PDF, Excel, on-screen preview, and Certinia payload continue to use the selected privacy settings.

## Safety

- No database schema change.
- No invoice is created.
- No invoice number is consumed.
- No SELL synchronization or cutover is enabled.
- No Certinia transmission is enabled or performed.
