# 026A CRM Integration Framework

## Purpose
Build a demo-ready CRM Integration Center that prepares ProjectPulse for Salesforce and Zendesk Sell integration.

## What It Shows
- Salesforce placeholder connector.
- Zendesk Sell placeholder connector.
- CRM opportunity/deal/quote mapping.
- CRM to SOW Generator workflow.
- CRM to Sales-to-Delivery Intake workflow.
- Sync readiness checklist.
- No-secrets control.

## Demo Flow
1. Open Dashboard.
2. Open CRM Integration Framework.
3. Review Salesforce and Zendesk Sell placeholders.
4. Review field mapping.
5. Review sync preview.
6. Complete readiness checklist.
7. Validate framework.
