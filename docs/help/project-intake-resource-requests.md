# Project Intake and Engineering Resource Requests

## Purpose

This module gives the August demo a working shell for project intake, project workspace readiness, engineering resource requests, and staffing visibility.

## Demo Scope

- View project intake queue.
- Create a new intake request.
- View project/workspace readiness.
- View engineering resource requests.
- Create a new engineering resource request.
- Show available engineers, skills, capacity, and assignment status.
- Keep assignment approval and Salesforce/Outlook integrations out of scope for now.

## Required Permissions

- `VIEW_PROJECT_INTAKE`
- `MANAGE_PROJECT_INTAKE`
- `VIEW_RESOURCE_SCHEDULING`
- `MANAGE_RESOURCE_SCHEDULING`
- `VIEW_ENGINEERING_RESOURCE_REQUESTS`
- `MANAGE_ENGINEERING_RESOURCE_REQUESTS`
