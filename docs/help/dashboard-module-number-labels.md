# Dashboard Module Number Labels

## Purpose
Adds consistent dashboard eyebrow labels so every Dashboard card displays `MODULE ###` instead of mixed category labels.

## Scope
This is a visual consistency patch only.

## Webpage impact
Dashboard card labels change from examples such as:
- TIMESHEET
- PROJECT WORKLOAD
- CUSTOMERS
- WORKFLOW

to:
- MODULE 001
- MODULE 018
- MODULE 021
- MODULE 007

## No behavior changes
This patch does not change:
- Routes
- Permissions
- Buttons
- Descriptions
- Backend APIs
- Module behavior
