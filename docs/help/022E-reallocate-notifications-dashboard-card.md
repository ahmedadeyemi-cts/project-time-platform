# 022E Reallocate Notifications to Dashboard Card

## Purpose
Move the Production Notification Center out of the top bar so it does not conflict with the global View As selector.

## Changes
- Hides the prior top-bar Notifications dropdown.
- Adds a dashboard card for Production Notification Center.
- Expands notifications inline from the dashboard card.
- Replaces problematic Readiness Center navigation with inline readiness status loading.
- Makes Open related page perform a deterministic action.
- Keeps email disabled.
