# Project Workspace and Engineering Documents

## Purpose

The Project Workspace gives project managers, coordinators, managers, and engineers a project-centered view of assignments, tasks, resource requests, and documents.

## Project Documents

SOW, GSD, quotes, proposals, architecture documents, order forms, and supporting documents may be uploaded from Project Intake, even when the intake originated from Salesforce.

Engineering-visible documents can be downloaded from the workspace.

## Timesheet Description Assistant Readiness

SOW and GSD documents can be flagged for future AI-assisted timesheet description context. The production implementation should keep this vendor-neutral and route through an approved AI provider configuration.
