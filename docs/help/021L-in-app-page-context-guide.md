# 021L In-App Page Context Guide

## Webpage Impact

Adds a visible page context guide across signed-in app pages.

The guide explains:

- Current page purpose.
- Backend process or endpoint supporting the page.
- What the user should validate on that page.

## Backend Support

No new backend endpoint is required. The guide documents existing backend support for visible webpage areas.

## What to Check on the Webpage

- Open any signed-in page.
- Confirm the guide appears near the top of the page.
- Confirm the guide changes when navigating between pages.
- Confirm it explains the page purpose, backend support, and what to check.
- Confirm Show / hide works.
